Aplicativos da Web App Service Azure

App Service Web Apps é uma plataforma de computação totalmente gerenciada otimizada para hospedagem de sites e aplicativos da web. Essa oferta de plataformas como serviço (PaaS) do Microsoft Azure permite que você se concentre em sua lógica comercial, enquanto a Azure cuida da infraestrutura para executar e dimensionar seus aplicativos.

Azure Content Delivery Network (CDN)

A Rede de entrega de conteúdo Azure (CDN) armazena conteúdo estático da Web em locais estrategicamente posicionados para fornecer a máxima taxa de transferência para entregar conteúdo aos usuários. O CDN oferece aos desenvolvedores uma solução global para fornecer conteúdo de alta largura de banda armazenando em cache o conteúdo em nós físicos em todo o mundo.
